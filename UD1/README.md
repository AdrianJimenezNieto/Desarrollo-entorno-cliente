# **Tarea evaluable 1**  

Esta es la primera tarea de la asignatura "Desarrollo en Entorno Cliente". En esta actividad, se trabajará sobre la selección de lenguajes utilizados en el entorno cliente, centrándose en HTML, CSS y JavaScript.  

El objetivo principal es comprender las diferencias entre estos lenguajes y analizar por qué son adecuados para la programación del lado cliente. A través de esta tarea, se busca introducir los conceptos básicos y la importancia de cada uno en el desarrollo web.
